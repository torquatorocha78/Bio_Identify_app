🧬 Bio Identify
Seu Biólogo de Bolso

Bio Identify é um aplicativo web desenvolvido em Python + Streamlit que utiliza IA multimodal para identificar animais, plantas, fungos e outros seres vivos a partir de uma foto enviada pelo usuário. O app retorna a identificação provável, classificação biológica básica e mantém um histórico local de identificações.

🚀 Funcionalidades

📸 Upload de imagens (jpg, png, jpeg)

🤖 Identificação automática do ser vivo via LLM

🧠 Retorno estruturado:

Nome comum

Nome científico

Reino / Filo / Classe (quando possível)

Descrição curta

🗂️ Histórico local das identificações (SQLite)

📊 Visualização dos registros no próprio app

☁️ Pronto para deploy no Streamlit Cloud

🧪 Tecnologias Utilizadas

Python 3.11

Streamlit

OpenAI API (modelo multimodal)

Pillow (PIL)

SQLite

Pandas

python-dotenv
